JSON

发表于 2017-03-09 | 分类于 Code |
  1. json字符串的使用必须是双引号，字符串变量‘ok’是错的。
  2. 属性键也必须使用双引号。
  3. JSON.stringify(value,replacer,space)把js对象变成json格式.第一个参数是要处理的值，第二个的值是要在做之后对值处理

  4. var student = new Object();
    student.name = “Lanny”;
    student.age = “25”;
    student.location = “China”;

    // var json = JSON.stringify(student);
    如果没这个函数的话，只是弹出来的时候还是[object,object]但是你找的时候还是很简答的啊。
    alert(student);

  5. var students = new Array() ;
    students[0] = “Lanny”;
    students[1] = “dong”;
    students[2] = “I love you”;
    var json = JSON.stringify(students,switchUpper);
    function switchUpper(key, value) {
    return value.toString().toUpperCase();
    }
    alert(json);

  6. parse用于从一个字符串中解析出json对象,如

    var str = ‘{“name”:”huangxiaojian”,”age”:”23”}’

    结果：JSON.parse(str)

  7. 可选参数replacer是用来替换参数Valued的。具体的替换规则是这样的：JSON.stringify({A:5,B:[9,2]},replacer);function replacer(key,value){
    if(typeof value==’number’){
    value=2*value;
    }
    return value;
    }
    “{“A”:10,”B”:[18,4]}”
  8. 当replacer为数组的时候，用来隐藏非数组对象内属性不在这个列表中的所有属性。JSON.stringify({foo:1,bar:{foo:1,bar:2}},[‘bar’]);—
    “{“bar”:{“bar”:2}}”
  9. 最后一个是用来影响输出格式的。如果没有这个参数，那么变换后的结果实单行的文本。有了这个可以可以插入新行而且通过数组和对象的嵌套增加缩进。有这两种方法可以指定如何缩进。
  10. 他的参数可以是什么呢？数字。。。将数字会成以相对应的数量的空格来缩进。
  11. 一个不被JSON支持的值将解析成为undefined,而不是string.
  12. 可以使用一个函数toJSON来先把这个转换为字符串，然后使用该方法来做JSON.stringify().
