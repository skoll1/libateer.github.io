Chrome调试

发表于 2017-03-09 |
  1. console.log()还可以用Jquery语法。
  2. cmd+o打开当前文件夹。
  3. 查找当前元素绑定的js文件。css也可以找到。
  4. 在console.log()中输入getEventListeners(a)可以获得当前节点的所有绑定的事件。
  5. 在Js文件中选中一段代码后，ctrl+shift+e可以在console.log()面板中运行这段代码。
  6. console.trace()可以追踪代码执行过程中的栈信息。
  7. timeline时间线如果出现红线就说明渲染时间超过了18ms，点击这些红色的线，就可以知道相应的警告信息，不超过16ms的才是流畅的画面。
  8. 快捷方法$()返回指定的css的选择器相匹配的的第一个元素。$$()返回与指定的css选择器相匹配的所有元素的数组。
  9. chrome第一个elements右边的那一行最后两个是干嘛的。
  10. cmd+o是寻找整个文件中你想要知道的东西的位置。
  11. cmd+f是当前文件夹内寻找。
  12. 如果你希望在所有加载的文件中搜索特定的文字，你可以用下面的快捷键来加载搜索框界面：cmd+ shift+f
  13. Cmd + Shitf + O 定位到特定的函数。
  14. network查看网络请求的全部信息。
  15. lnitiator:parser:请求是html发起的。script：脚本。redirect:页面重定向发起的。other:其他发出的。
  16. Cmd + K 清除控制台
  17. Shitf + Enter 允许你从控制台进入多行模式。
  18. ###console.log()API###
  19. console.trace()跟踪一条栈信息。
  20. console.assert(blloean,obj):如果表达式错误，就把obj打印
  21. console.count()记录当前被执行的次数。
