css-基础

发表于 2017-03-10 | 分类于 Code |
  1. 小东西：淘宝，搭配，比较-游戏-社交-好的一部分-商品评论
  2. 你给我一个目标，我会马上调动我的知识库帮你完成这个，扩展，用某种观点。
  3. 字体–font-style:让字体变斜就这个。
  4. font-weight:粗细就看这个
  5. font-stretch:chrome不支持
  6. @font-face{
    font-family:’iconfont’;
    src:url(‘’);
    }引入字体。
  7. font-varight:今年仅针对字母：小型的大写字母。
  8. text-transform:也是和字母有关系，将第一个值变成大写字母，全变成大写，全变成小写。
  9. white-space:和文字空间有关系的。pre:不会合并文字间的空白，超出边界不换行只会显示一行。nowrap:合并文字间空格，只会显示一行。pre-wrap:不合并空格，但是会发生换行碰到边界。pre-line:会合并空格，也会换行。
  10. word-space:单词之间的间距,在这个测试里好像他完全是以空格来区分单词的，不论是汉字还是英文。
  11. letter-space:字符之间的间距
  12. text-indent:文本缩进。
  13. word-break:break-all;word-wrap:break-word;想要节省地方，这么写就对了。
  14. text-align-last:只对最后一行做这个属性设置
  15. text-justify:类似于flex那种对齐方式，但是没什么卵用，好多浏览器不支持。
  16. vertical-align:他的百分比是按照line-height计算的。
  17. vertain—align：起作用的前提是他的元素是inline属性才行。inline-block.table-cell()简单来说就是图片，按钮，文字和单元格。这只是在默认的情况下，当给他设置属性的时候或者使用浮动以后，那就不能显示了。`.test>span{
        display: inline-block;width: 210px;vertical-align: middle;
}
.test>img{
    vertical-align: middle;
    width: 300px;
    height: auto;
}`
  18. text-size-adjust:auto;文字根据设备大小自动调整
  19. text-decoration:怎么改下划线加颜色啊。text-decoration-color: #f06d06;这个好像不能显示，只能和color共用一个颜色。
  20. writing-mode:实现文字的排列
  21. 只要你是内联元素，你就受到vertain-align和line-height的影响。
  22. 行高是字符大小乘百分比。
  23. 定位position:staticabsolute,relative,center,fixed;
  24. 布局：display:inline,block,flex,inline-felx,table
  25. display:none;彻底消失，原来是位置也腾出来。visibilty:hidden.就是简单的影藏，原来的位置还在。
  26. margin:相邻的margin会发生合并，我们称之为margin折叠。折叠只会发生在block元素。浮动元素的margin不发生任何折叠。根元素，绝对定位的元素也不发生折叠。
