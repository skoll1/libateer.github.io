css-基础

发表于 2017-03-10 | 分类于 Code |
  1. 小东西：淘宝，搭配，比较-游戏-社交-好的一部分-商品评论
  2. 你给我一个目标，我会马上调动我的知识库帮你完成这个，扩展，用某种观点。
  3. 字体–font-style:让字体变斜就这个。
  4. font-weight:粗细就看这个
  5. font-stretch:chrome不支持
  6. @font-face{
    font-family:’iconfont’;
    src:url(‘’);
    }引入字体。
  7. font-varight:今年仅针对字母：小型的大写字母。
  8. text-transform:也是和字母有关系，将第一个值变成大写字母，全变成大写，全变成小写。
  9. white-space:和文字空间有关系的。pre:不会合并文字间的空白，超出边界不换行只会显示一行。nowrap:合并文字间空格，只会显示一行。pre-wrap:不合并空格，但是会发生换行碰到边界。pre-line:会合并空格，也会换行。
  10. word-space:单词之间的间距,在这个测试里好像他完全是以空格来区分单词的，不论是汉字还是英文。
  11. letter-space:字符之间的间距
  12. text-indent:文本缩进。
  13. word-break:break-all;word-wrap:break-word;想要节省地方，这么写就对了。
  14. text-align-last:只对最后一行做这个属性设置
  15. text-justify:类似于flex那种对齐方式，但是没什么卵用，好多浏览器不支持。
  16. vertical-align:他的百分比是按照line-height计算的。
  17. vertain—align：起作用的前提是他的元素是inline属性才行。inline-block.table-cell()简单来说就是图片，按钮，文字和单元格。这只是在默认的情况下，当给他设置属性的时候或者使用浮动以后，那就不能显示了。`.test>span{
        display: inline-block;width: 210px;vertical-align: middle;
}
.test>img{
    vertical-align: middle;
    width: 300px;
    height: auto;
}`
  18. text-size-adjust:auto;文字根据设备大小自动调整
  19. text-decoration:怎么改下划线加颜色啊。text-decoration-color: #f06d06;这个好像不能显示，只能和color共用一个颜色。
  20. writing-mode:实现文字的排列
  21. 只要你是内联元素，你就受到vertain-align和line-height的影响。
  22. 行高是字符大小乘百分比。
  23. 定位position:staticabsolute,relative,center,fixed;
  24. 布局：display:inline,block,flex,inline-felx,table
  25. display:none;彻底消失，原来是位置也腾出来。visibilty:hidden.就是简单的影藏，原来的位置还在。
  26. margin:相邻的margin会发生合并，我们称之为margin折叠。折叠只会发生在block元素。浮动元素的margin不发生任何折叠。根元素，绝对定位的元素也不发生折叠。
  27. border-首先是我们熟悉的那一套，还有就是border-image:

  28. `width: 300px;

    height: 300px;
border: 30px red solid;
<!--必须有这个-->
border-image-source:url(1.jpg);
border-image-slice: 27;
border-image-width: 27px 1 10% 27px;
border-image-outset:27px;
/*border-image-repeat: stretch;*/
/*border-image-repeat: repeat;*/
/*border-image-repeat: round;*/
border-image-repeat: space;`

  29. background:可以加纯色。图片。渐变。

  30. background-repeat:round;自动缩放，会有repeat效果;space：没有任何效果。
  31. background-attachment:fixed; scroll,local都不动
  32. background-position:数字，关键字。center center;
  33. 但是background-origin:padding-box,content-box,border-box;只有这几个关键字能用。
  34. background-size:100% 100%这个会强行把他放大到全部包含背景,cover:等比例覆盖：可能会超出容器 ,contain:缩放，但是始终在容器内.在满足一个方向之后就不管了。。。;
  35. background-clip的真正作用是决定背景在哪些区域显示。如果它的值为border,则背景在元素的边框、补白和内容区域都会显示；如果值为padding,则背景只会在补白和内容区域显示；如果值为content，则背景只会在内容区域显示。
  36. background-origin是指背景显示的区域，或者说背景是从哪个区域开始绘制的(边框、补白或内容区域)。
  37. 该属性指定了背景在哪些区域可以显示，但与背景开始绘制的位置无关，背景的绘制的位置可以出现在不显示背景的区域，这时就相当于背景图片被不显示背景的区域裁剪了一部分一样。
  38. 该属性指定了背景从哪个区域(边框、补白或内容)开始绘制,但也仅仅能控制背景开始绘制的位置，你可以用这个属性在边框上绘制背景，但边框上的背景显不显示出来那就要由background-clip来决定了
  39. filter:属性，blur(设置高斯模糊)，brightness(%调节亮度)，contrast(%图片对比度)，drop-shadow(类似于盒子阴影),grayscale(%)定义图片灰度。hue-rotate(deg定义色相翻转),invert(%翻转输入图像)，saturate(%定义图像饱和度)，
  40. list-style-type:
  41. caption:默认的是在上边，而且他还不属于table里面。
  42. caotion-side:boottom,top,left;放在左边是不得先排那个字的走向
  43. empty-cells: show;表格无内容的时候，不显示边框。
  44. border-collapse: collapse;边框之间的边框省略。
  45. content:尾类中的使用；none,attr,url,string;
  46. 所有主流浏览器都不支持 appearance 属性。为什么这样？
  47. 浏览器对每个html元素的渲染都是不一样的，比如button,在chrome中，渲染出来的效果是不一样的。我们可以用appearance属性对任何元素的渲染风格改变，最简单的来说，我们可以把‘a’连接默认显示成“按钮”风格，那么我们可以使用‘appearance’设置为“button”.
  48. text-overflow:clip(超出部分裁掉)，ellipsis(替换为…);
  49. outline:设置或检查对象外的线条轮廓。
  50. cursor:鼠标各种的抚摸特效。
  51. zoom:设置对象缩放比例。可以缩放任何东西。
  52. user-select:none(文本不能被选择)，text(可以),all(将所有内容变成一个块的时候可以)，element(可以选，但是选择范围受限制)
  53. resize:如果希望此属性生效，需要设置对象的overflow属性.both.horizontal,vertical.
  54. transfrom:translate(),scale(),rotate(),skew()斜切扭曲。
  55. 3D转换需要设置的新属性：perspective,transform-style:flat,perserve-3d;perspective-origin:指定透视点的位置。
  56. backface-visibility:visible,指定背面元素可见。hidden：指定元素背面不可见。
  57. transition:注意可以起作用的对象，z-index.
  58. animation: name 持续时间 过渡帧率 延迟时间 循环次数(1,infinite无限次) 动画的播放方向 默认向前播放，alternate（偶次向前，基数向前）
  59. media引入的两种方式：

  60. @media screen and (min-width:600px)and (max-width: 1000px){

        body {
        background-color:lightblue;
}

    }小的必须在前面，错了，跟这个没关系!有关系的是必须要在他们之间留下足够的空格。

  61. 选择所有作为E元素的子元素F E>F适用于剔除孙子元素里面不想选的，但是子元素里面有的。也就是说孙子和儿子都有的东西，但是你只想选择儿子的。

  62. 选择紧贴在E元素之后F元素。 E+F p+p的选择就是先必须找到两个p,然后后一个被选中。
  63. 选择E元素后面的所有兄弟元素F。 E~F p~p 表示只要出现第一个p元素之后，接下来只要有p元素都会起到作用的
  64. 选择中的标签属性，这个是来动态变化的添加css属性。
  65. img[type=”text”] img[type^=”text”] img[type$=”text”] img[type*=”text”] img[type|=”text”]开头具有用”-“分隔的字符串。
  66. link visited hover active focus(成为输入焦点的时候)
  67. 还可以筛选文字形式给确定文字样式。
  68. border:none;border:0;的区别border:none; => border-style: none;border:0; => border-width: 0;两者理论上都是一个效果，就是不显示边框实际上，我记得 none 在某个 ie 下面有 bug，用 0 没有 bug。
    至于 border:0 none; 没必要;后一个是小号内存了。
