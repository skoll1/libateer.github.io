飞鸟 若非我是一只鸟， 不然， 翅膀的每一次划动， 怎么， 那么令人迷醉。 轻轻的，轻轻的 和着那揉碎在风中的思念， 飞翔，荡漾， 等待着人们自由的 挑选。 在暴风雨来临的时候， 伴着雨点， 抖动羽毛， 来伴奏天地间的旋律。 却一不小心把思念吹走了。 慢慢，慢慢， 世界将我遗忘， 这只生长在风中的鸟， 这只连睡觉都在风中的鸟， 到了落地的时候了。 睁开了迷蒙的双眼， 在某个晴朗的早上， 我看到世界在向我点头， 仿佛让我跟随他， 第一次，我把目光投向了天空。 为什么只能看到太阳， 却听不到他的声音呢。 第一次，我看到了几千个太阳一起升起的情景。

  1. 最是一池春皱处，西湖边下无人家
  2. 捻碎 花开气候 朦胧 谁惹 清波 皱
  3. 即使活着、也极其孤独的他，无法把那样的他留在那里。我的生涯中就这一次，要只想着我自己，要为我自己而活。我会接受惩罚的。——宋恩彩
  4. 生活在太耀眼的人身边，就像照耀着一颗太阳，你会热到忘了自己的温度，一个人一生只做了一次选择，那就是陪着他去死。
