  1. meta标签的组成：meta标签共有两个属性，他们非别是http-equiv属性和name属性，不同的属性有不同的参数，这些不同的参数值实现了网页不同的功能。
  2. name主要用于描述网页，与之对应的属性值为content,content的内容主要是便于搜索引擎机器人查找信息和分类信息。
  3. Robots:告诉机器人那些页面需要索引，哪些不需要。。all:文件将被检索，而且页面上的链接可以被查询,none：文件不被查询，链接也不能被查询,index:文件将被检索,follow:页面上的链接可以被查询,nofollow：文件将被检索，但是页面上的链接不可以被查询.noindex:文件不能被检索，但是链接可以被查询。
  4. Generator:代表说明网站采用什么软件制作。
  5. Copyright:代表网站的版权信息。
  6. Revisit-after:7days:表示网站重访问的天数。
  7. Http-equiv:expires:期限：用于设置网页的到期时间，一旦网页过期，必须到服务器上重新传输。必须使用GMT的时间格式。
  8. Pragma:禁止浏览器从本地计算机的缓存中访问页面内容。：no-cache.
  9. Refresh:”2;URL=http://baidu.com"自动刷新并且指向新页面。这个的意思是停留两秒自动刷新新的URL网址。
  10. Set-Cookie:如果网页过期，那么存盘的cookie将被删除。cookie value=*,expires=
  11. Window-target:显示窗口的设定，强制页面在当前页面以独立页面显示：用来防止别人在框架里调用自己的页面。
  12. Content-type:显示字符集的设定：设定页面使用的字符集。
  13. Content-langue:显示语言的设定。
  14. Cache-control:指定请求和响应遵循的缓存机制。
  15. overFlow:scroll的时候不论是否需要，用户都会出现滚动机制。
  16. 新窗口打开要用那个值_blank

###深入了解font-weight:

  1. 字体的所拥有的字重输血量实际上很少存在满足9个字重的css一一对应的情况，通常有的为4-6个。
  2. 通常情况下，一个特定的字体仅仅会包含少数可用的字重，这就是我们我们有时候使用特定的字重没有“生效”，看起来跟其他字重差不多的原因所在

###overflow### :

###position###

  1. absolute；位置跟随，和原来的位置一样，去除浮动，去就是会说，绝对定位和float只能存在一个一次。
  2. float：属于布局属性。
  3. absolute与整体布局：body{position:absolute;l0;r0;t0;b0} html,body{h100%}

###line-height###

  1. 内联元素的高度由行高决定，而不是字体大小决定。
  2. 内容区域+行间距=行高。
  3. 内容区域：字号和字体决定。

###relative###

  1. relative可以限制absolute的left,right还可以限制层级，以及overflow;
  2. relative定位是相对自身的，无侵入性，他自己玩，不影响别人。
  3. 绝对定位是拉伸，相对定位是竞争，先left,top。
  4. 如果要使用relative，最好单独包在一起实现。
  5. 尽量避免使用relative. absolute定位不需要依赖relative.

  8. 页面根元素天然具有层叠上下文关系。

###css专业技巧###

  1. 优化技巧：避免使用一些表达式以及Filter()属性，因为每次的操作系统都需要重新绘制图像。css缩写。选择器尽量简洁。利用好继承机制。

  2. css继承机制。一个选择器的具体特殊性如下确定：

    对于内联样式为：1，0，0，0。 对于选择器中的ID属性值，加0，1，0，0。 对于选择器中的类属性值、属性选择、或伪类，加0，0，1，0。 对于选择器中的元素和伪元素，加0，0，0，1。 通配符选择器（*），加0，0，0，0。

    h1 {color:red;}specificity=0,0,0,1 pem{color:purple;}/*specificity=0,0,0,2*/ .grade{color:purple;}/*specificity=0,0,1,0*/ *.bright{color:yellow;}/*specificity=0,0,1,0*/ div#header [href]{color:black;}/*specificity=0,1,1,1 包涵一个属性选择器*/

可以继承的属性以下的样式：所有关于文字图片大小样式的属性可以继承( 例如：letter-spacing、word-spacing、white-space、line-height、color、font等)

层叠样式表中最基本的方面就是层叠——冲突的声明要通过这个层叠过程排序。

读者的重要声明（！important）>创作人员的重要声明(! important)>内联样式（1，0，0，0）>id选择器（0，1，0，0）>类选择器、属性选择器、伪类选择器(0,0,1,0)>元素选择器(0,0,0,1)>

通配符选择器（0，0，0，0）>继承的属性>浏览器默认样式。

###浮动的原理和工作方式###

  1. 浮动元素脱离文档流，不占据空间。浮动元素碰到包含他的边框或者浮动元素的边框停留。
  2. 影响：浮动元素自动变成块状元素。导致父元素无法被撑开，与该浮动元素同级的非浮动元素，如果是块级的话，那么就移动到钙元素的下方，如果是内联元素则会环绕该浮动元素。与该元素同级的浮动元素，对于同一个方向的浮动元素，两个元素将会跟在碰到的浮动元素的后面，但是不同方向的浮动，在宽度足够的时候，将分别浮动想不同的方向，在宽度不同的时候，将导致一方换行（换行与html书写顺序有关，后边的将会浮动到下一行。）
  3. 清除浮动：空标签，overflow,
  4. .outer{border: 1px solid #ccc;background: #fc9;color: #fff;padding: 50px;margin:50px auto;/*overflow: auto;*/ /*这是什么原理？*/ } .outer :after {clear:both;content:'.';display:block;width: 0;height: 0;visibility:hidden;} /*本来在浮动的影响下什么都不会显示的的，但是添加了padding就会显示内部背景颜色*/ .div1{width: 80px;height: 80px;background: red;float: left;} .div2{width: 80px;height: 80px;background: blue;float: left;} .div3{width: 80px;height: 80px;background: sienna;float: left;} /*如果没有添加浮动：最外层的父级元素不能显示背景颜色，边框不能撑开，margin值不能正确显示。*/ /*.clear{ clear:both; }*/清除浮动的所有代码来了。
